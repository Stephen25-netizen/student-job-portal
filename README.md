# Job Portal

A responsive job search portal with search functionality.

## Features
- Real-time job search by title, company, location, experience
- Responsive design for mobile/desktop
- Hover animations on job cards
- Clean modern UI

## Live Demo
Will be available at: https://YOUR_USERNAME.github.io/job-portal after deployment

## How to Run Locally
1. Open `index.html` in any browser
2. Type in search bar (e.g. "Engineer", "Nairobi", "3+")

## Tech Stack
- HTML5 Semantic
- CSS3 Flexbox + Animations
- Vanilla JavaScript (no frameworks)

## Deployment
Deployed to GitHub Pages for free hosting with custom domain support.
